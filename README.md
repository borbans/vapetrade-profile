# VapeTrade – E-Commerce for Vaping Products

This repository provides public information about **VapeTrade.de**, a German online shop for e-cigarettes, liquids, aromas and accessories.

## About VapeTrade

VapeTrade.de focuses on:

- Modern e-cigarettes (pod systems, starter kits, box mods)
- Longfill & shortfill aromas
- Nic salt liquids & 10 ml liquids
- Coils, tanks, pods and accessories
- Tools and guides for vapers (calculator, blog, FAQs)

The goal is to combine:

- Transparent product data
- Strong technical SEO (schema.org, JSON-LD)
- Fast and mobile-optimized UX
- Clear information for beginners and advanced users
